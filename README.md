# 项目简介

## 核心功能
- 基于YOLOv8的目标区域摔倒检测报警系统
- 支持图片、视频及摄像头进行检测
- 支持选择模型
- 界面可实时显示目标位置、目标总数、置信度等信息
- 支持批量检测在界面直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择检测模型
- 支持查看批量检测每一张检测结果
- 摔倒区域检测，报警设置，记录图像

## 适用场景
- 需要实时摔倒检测的安全监控场景
- 老人、儿童等特殊群体的安全监护
- 医疗机构、养老院等场所的安全保障
- 需要自动化检测跌倒事件的环境监测

## 技术特点
- 使用Python 3.9、OpenCV、PyQt5、Ultralytics、Torch1.9等技术开发
- 包含1000张标注数据集，分为'Fall Detected'和'NoFall Detected'两类
- 预训练模型可直接使用，配置环境后即可运行
- 提供完整的UI界面源文件和图标
- 支持多种输入源检测，包括单张图片、图片文件夹、视频文件和本地摄像头
